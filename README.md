# Data Loss Prevention using Two-Factor Authentication and Role-Based Access Control
## Project Overview
This academic project focuses on enhancing data security in multi-user environments by implementing Two-Factor Authentication (2FA) and Role-Based Access Control (RBAC). The system ensures that only authorized users can access sensitive data, reducing the risk of unauthorized access and data leakage.
## Features
- User Registration and Login
- Email OTP-based Two-Factor Authentication (2FA)
- Role-Based Access Control (Admin and User)
- Secure Data Access
- Authentication and Authorization
- Enhanced Data Protection
